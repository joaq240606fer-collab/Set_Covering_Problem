# Optimización de Cobertura de Estaciones de Radio
### Set Covering Problem con Algoritmo Greedy
--- 
##  Descripción
Este proyecto resuelve un **Set Covering Problem** clásico: seleccionar el número mínimo de estaciones de radio necesarias para cubrir todos los estados del oeste y centro de Estados Unidos.  

Inspirado en el capítulo 8 del libro:

**Grokking Algorithms – Aditya Bhargava**

https://github.com/egonSchiele/grokking_algorithms.git

---

##  Objetivo
- Conjunto de estados a cubrir
- Estaciones que cubren conjuntos de estados
- Algoritmo **greedy** que selecciona la estación que cubre más estados aún no cubiertos
- Obtener un conjunto mínimo aproximado de estaciones
